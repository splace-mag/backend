Important! It seems that your browser is not able to load some of the resources needed to run this site. To overcome this please use Chrome or Safar or served this site through a webserver. Your widget is going to be loaded through an AJAX web call. These calls are restricted by a security manager that will only allow loading resources from a common URL.

An easy to use web server for Windows is available in the XAMPP project.

This window above is exactly 1024x730 px large which is going to be roughtly the viewport we are going to have on an iPad.

Use the "load" button above to load and the "remove" to remove your widget from the main framework.

Place your widget in the folder "app/" all files named according to the following naming conventions.

If your app is named dummy you will have "dummy.html", "dummy.css", and "dummy.js".
